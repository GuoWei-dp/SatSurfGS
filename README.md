# SatSurfGS

SatSurfGS is an implementation of the paper SatSurfGS: Generalizable 2D Gaussian Splatting for Sparse-View Satellite Surface Reconstruction.




Note：This repository is undergoing revisions, and our paper is still under review. A preprint of this article has been published in arXiv, arXiv ID: arXiv: 2605.07181.
